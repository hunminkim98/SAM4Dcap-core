# webviz

## Purpose
This directory contains the clean OpenCap motion viewer.

## Key Files
| File | Description |
|------|-------------|
| `index.html` | Viewer page. |
| `main.js` | Runtime loader for motion scenes. |
| `style.css` | Viewer styling. |

## Subdirectories
| Directory | Purpose |
|-----------|---------|
| `Geometry/` | OpenSim mesh assets used by the viewer. |

## For AI Agents

### Working In This Directory
- Keep the viewer zero-build and directly servable.
- Do not rename mesh assets without updating the scene JSON that references them.

### Testing Requirements
- Serve the parent `web/` directory and verify motion playback and mesh loading.

### Common Patterns
- Static site with adjacent geometry assets.

## Dependencies

### Internal
- Scene JSON is generated by the clean workflow and/or export helpers.

### External
- Browser-side rendering and OpenSim VTP geometry.
